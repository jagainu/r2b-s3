# 要件定義書: ホーム画面

## 目的

アプリのエントリーポイント。クイズを開始する起点となる画面。今日の一匹をクイズ形式で提示し、毎日の学習習慣を促す。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | クイズ開始 | クイズ画面（P002）へ遷移してクイズを開始する | 新規 |
| 2 | 今日の一匹 | ユーザーごとに1日1回変わるランダムな猫をクイズ形式で出題する | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| クイズ開始 | 「クイズを始める」ボタンタップ | セッション状態を初期化 | P002（クイズ画面）へ遷移 |
| 今日の一匹 | 現在日付・ユーザーID（端末識別子） | 日付＋端末IDをシードにランダム猫を1種選択。同日中は同じ猫を返す | 猫の写真＋4択クイズを表示 |
| 今日の一匹（回答） | ユーザーの選択肢タップ | 正解判定・回答履歴を端末に保存 | P003（解説画面）へ遷移 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| 「クイズを始める」ボタン | クイズ開始 | 通常 / タップ中（押下エフェクト） |
| 今日の一匹エリア | 今日の一匹 | ローディング中 / 未回答（クイズ表示） / 回答済み（結果表示） |
| 選択肢ボタン × 4 | 今日の一匹（回答） | 通常 / タップ中 / 正解（緑） / 不正解（赤） |
