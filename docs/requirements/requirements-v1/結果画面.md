# 要件定義書: 結果画面

## 目的

10問のクイズ完了後に表示される画面。正答率・覚えた種類数を表示し、継続学習を促す。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | 結果表示 | 今回セッションの正答率・正解数・不正解数を表示する | 新規 |
| 2 | 累計表示 | 端末（localStorage）に保存された累計の覚えた種類数を表示する | 新規 |
| 3 | もう一度挑戦 | ホーム画面（P001）へ戻る | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 結果表示 | セッションの正解数・不正解数 | 正解数 / 10 で正答率を計算 | 正答率・正解数・不正解数を表示 |
| 累計表示 | localStorage の誤答履歴・正答履歴 | 一度でも正解した猫種のユニーク数をカウント | 累計覚えた種類数を表示 |
| もう一度挑戦 | 「もう一度挑戦」ボタンクリック | - | P001（ホーム画面）へ遷移 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| 正答率 | 結果表示 | 表示中 |
| 累計覚えた種類数 | 累計表示 | ローディング中 / 表示中 |
| 「もう一度挑戦」ボタン | もう一度挑戦 | 通常 / クリック中 |
