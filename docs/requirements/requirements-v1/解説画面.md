# 要件定義書: 解説画面

## 目的

クイズ回答後に表示される画面。正解・不正解に関わらず猫の特徴と似た種類を提示し、「なぜこの種類か」を理解することで定着を促す。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | 正解・不正解表示 | 回答が正解か不正解かを視覚的に表示する | 新規 |
| 2 | 猫の特徴表示 | 正解の猫の写真・種類名・毛色・模様・毛の長さを表示する | 新規 |
| 3 | 似た種類表示 | 見た目が似ている猫の種類名＋写真を一覧表示する | 新規 |
| 4 | 次の問題へ | 次の問題（クイズ画面）または完了後にホームへ遷移する | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 正解・不正解表示 | 直前クイズの正解フラグ | フラグを評価 | 正解バナー（緑）または不正解バナー（赤）を表示 |
| 猫の特徴表示 | 正解の猫ID | 猫IDをキーに猫データ（写真・種類名・毛色・模様・毛の長さ）を取得 | 猫の写真・種類名・特徴リストを表示 |
| 似た種類表示 | 正解の猫ID | 手動登録の類似猫リスト＋毛色・模様・毛の長さが1つ以上一致する猫を自動抽出（手動優先） | 似た猫の種類名＋写真を最大3件表示 |
| 次の問題へ | 「次の問題へ」ボタンクリック・現在の問題番号 | 問題番号が10未満 → 次の問題へ。10問完了 → セッション結果を端末（localStorage）に保存 | P002（クイズ画面）または P006（結果画面）へ遷移 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| 正解・不正解バナー | 正解・不正解表示 | 正解（緑・「正解！」） / 不正解（赤・「不正解」） |
| 猫の写真 | 猫の特徴表示 | ローディング中 / 表示中 |
| 似た種類エリア | 似た種類表示 | 表示中 / 該当なし（非表示） |
| 「次の問題へ」ボタン | 次の問題へ | 通常 / クリック中。10問完了時はラベルが「結果を見る」に変化 |
