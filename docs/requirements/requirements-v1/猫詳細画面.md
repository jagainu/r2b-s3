# 要件定義書: 猫詳細画面

## 目的

特定の猫種の詳細情報を確認できる画面。複数の写真をスワイプで閲覧でき、毛色・模様・毛の長さの特徴を確認できる。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | 写真切り替え | 複数枚の猫の写真をスワイプで閲覧する | 新規 |
| 2 | 特徴表示 | 種類名・毛色・模様・毛の長さを表示する | 新規 |
| 3 | 戻る | P004（猫図鑑画面）へ戻る | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 写真切り替え | 遷移時の猫ID | 猫IDをキーに複数枚の写真URLリストを取得 | 写真カルーセルを表示（矢印ボタンで切り替え可能） |
| 特徴表示 | 遷移時の猫ID | 猫IDをキーに種類名・毛色・模様・毛の長さを取得 | 種類名・特徴リストを表示 |
| 戻る | 戻るボタンタップ | 遷移スタックを1つ戻る | P004（猫図鑑画面）へ遷移 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| 写真カルーセル＋矢印ボタン | 写真切り替え | ローディング中 / 表示中（現在枚数インジケーター付き） |
| 種類名・特徴リスト | 特徴表示 | ローディング中 / 表示中 |
| 戻るボタン | 戻る | 通常 / タップ中 |
