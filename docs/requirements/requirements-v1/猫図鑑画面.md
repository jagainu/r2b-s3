# 要件定義書: 猫図鑑画面

## 目的

全猫種を一覧で確認できる画面。毛色・模様・毛の長さのAND検索で絞り込みができ、詳細確認へ誘導する。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | 猫種一覧表示 | 全猫種をリスト形式（名前＋小さな写真）で表示する | 新規 |
| 2 | フィルタリング | 毛色・模様・毛の長さのドロップダウンでAND検索絞り込みをする | 新規 |
| 3 | 詳細表示 | 猫をタップしてP005（猫詳細画面）へ遷移する | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 猫種一覧表示 | 画面表示イベント | 全猫種データを取得 | リスト形式（名前＋サムネイル写真）で全件表示 |
| フィルタリング | 毛色・模様・毛の長さのドロップダウン選択値（各任意） | 選択された条件をANDで組み合わせて猫種データを絞り込む。未選択の条件は無視 | 絞り込み結果のリストをリアルタイム更新 |
| 詳細表示 | 猫リストアイテムのタップ・選択猫ID | 選択猫IDを遷移先に渡す | P005（猫詳細画面）へ遷移 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| フィルタードロップダウン × 3 | フィルタリング | 未選択（全件表示） / 選択中（絞り込み中） |
| 猫種リスト | 猫種一覧表示・フィルタリング | ローディング中 / 一覧表示 / 絞り込み結果表示 / 0件（「該当なし」メッセージ表示） |
| 猫リストアイテム | 詳細表示 | 通常 / タップ中 |
