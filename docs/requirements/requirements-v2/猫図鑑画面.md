# 要件定義書 v2: 猫図鑑画面

## 目的

全猫種を一覧で確認できる画面。毛色・模様・毛の長さのAND検索で絞り込みができ、詳細確認へ誘導する。

## 機能

| # | 機能 | 説明 | 対応DBテーブル |
|---|------|------|--------------|
| 1 | 猫種一覧表示 | 全猫種をリスト形式（名前＋サムネイル写真）で表示する | `cat_breeds`, `cat_photos` READ |
| 2 | フィルタリング | 毛色・模様・毛の長さのドロップダウンでAND検索絞り込みをする | `coat_colors`, `coat_patterns`, `coat_lengths`, `cat_breeds` READ |
| 3 | 詳細表示 | 猫をクリックしてP005（猫詳細画面）へ遷移する | なし（IDを遷移先に渡すのみ） |

## Input / Process / Output

| 機能 | Input | Process | Output | 対応DBテーブル |
|------|-------|---------|--------|--------------|
| 猫種一覧表示 | 画面表示イベント | `cat_breeds` を全件取得。各猫種の代表写真（`cat_photos.display_order = 1`）を JOIN して取得 | リスト形式（名前＋サムネイル写真）で全件表示 | `cat_breeds` READ, `cat_photos` READ |
| フィルタリング | 毛色・模様・毛の長さのドロップダウン選択値（各任意） | 選択された条件を AND で組み合わせて `cat_breeds` を絞り込み（`coat_color_id`, `coat_pattern_id`, `coat_length_id` で WHERE）。未選択の条件は無視 | 絞り込み結果のリストをリアルタイム更新 | `cat_breeds` READ |
| 詳細表示 | 猫リストアイテムのクリック・選択猫ID | 選択猫IDを遷移先に渡す | P005（猫詳細画面）へ遷移 | なし |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| フィルタードロップダウン × 3 | フィルタリング | 未選択（全件表示）/ 選択中（絞り込み中） |
| 猫種リスト | 猫種一覧表示・フィルタリング | ローディング中 / 一覧表示 / 絞り込み結果表示 / 0件（「該当なし」メッセージ表示） |
| 猫リストアイテム | 詳細表示 | 通常 / クリック中 |

## 補足

- フィルタードロップダウンの選択肢は `coat_colors`, `coat_patterns`, `coat_lengths` マスターテーブルから取得
- フィルタリングはクライアントサイドで実装してもよい（初回に全件取得済みのため）
- 写真は CloudFront 経由のURLを使用（S3直接参照は不可）
