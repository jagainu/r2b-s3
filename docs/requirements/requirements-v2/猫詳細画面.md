# 要件定義書 v2: 猫詳細画面

## 目的

特定の猫種の詳細情報を確認できる画面。複数の写真を矢印ボタンで切り替えながら閲覧でき、毛色・模様・毛の長さの特徴を確認できる。

## 機能

| # | 機能 | 説明 | 対応DBテーブル |
|---|------|------|--------------|
| 1 | 写真切り替え | 複数枚の猫の写真を矢印ボタンで切り替えながら閲覧する | `cat_photos` READ |
| 2 | 特徴表示 | 種類名・毛色・模様・毛の長さを表示する | `cat_breeds`, `coat_colors`, `coat_patterns`, `coat_lengths` READ |
| 3 | 戻る | P004（猫図鑑画面）へ戻る | なし |

## Input / Process / Output

| 機能 | Input | Process | Output | 対応DBテーブル |
|------|-------|---------|--------|--------------|
| 写真切り替え | 遷移時の猫ID | 猫IDをキーに `cat_photos` から写真URLリストを `display_order` 昇順で取得。矢印クリックで表示インデックスを更新 | 写真カルーセルを表示（矢印ボタンで切り替え可能・現在枚数インジケーター付き） | `cat_photos` READ |
| 特徴表示 | 遷移時の猫ID | 猫IDをキーに `cat_breeds` JOIN `coat_colors`, `coat_patterns`, `coat_lengths` を取得 | 種類名・毛色・模様・毛の長さの特徴リストを表示 | `cat_breeds` READ, `coat_colors` READ, `coat_patterns` READ, `coat_lengths` READ |
| 戻る | 戻るボタンクリック | 遷移スタックを1つ戻る | P004（猫図鑑画面）へ遷移 | なし |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| 写真カルーセル＋矢印ボタン | 写真切り替え | ローディング中 / 表示中（現在枚数インジケーター付き）/ 写真が1枚の場合は矢印非表示 |
| 種類名・特徴リスト | 特徴表示 | ローディング中 / 表示中 |
| 戻るボタン | 戻る | 通常 / クリック中 |

## 補足

- 写真と特徴は同一猫IDで取得するため、1回のAPIコールでまとめて返してよい
- 写真は CloudFront 経由のURLを使用（`cat_photos.photo_url` に CloudFront URL を格納）
- `display_order` の昇順で写真を並べる（display_order = 1 が最初に表示）
