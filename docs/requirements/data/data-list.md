## エンティティ関係サマリー

```
users
  ├── wrong_answers (1:N)
  ├── correct_answers (1:N)
  └── session_results (1:N)

cat_breeds
  ├── coat_colors (N:1)
  ├── coat_patterns (N:1)
  ├── coat_lengths (N:1)
  ├── cat_photos (1:N)
  ├── similar_cats (1:N 双方向)
  ├── wrong_answers (1:N)
  └── correct_answers (1:N)
```

## 補足

- 猫データ（cat_breeds・cat_photos・マスター）はシードデータとして初期投入
- 写真はS3に保存し、URLをDBに持つ
- 類似猫の自動抽出ロジック：毛色・模様・毛の長さが1つ以上一致する猫（手動登録を優先）
