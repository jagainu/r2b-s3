# 仕様書: ログイン・登録画面

## 目的

ユーザーがアカウントを作成またはログインする画面。認証後にホーム画面へ遷移する。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | メール＋パスワードログイン | メールアドレスとパスワードでログインする |
| 2 | Google ログイン | Google OAuth でログインする |
| 3 | 新規登録 | メールアドレス・パスワード・ユーザー名でアカウントを作成する |
| 4 | フォーム切り替え | ログインフォームと新規登録フォームを切り替える |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| メールアドレス入力欄 | ログイン・登録共通 | フォーム上部 |
| パスワード入力欄 | ログイン・登録共通 | メール入力欄下 |
| ユーザー名入力欄 | 新規登録時のみ表示 | パスワード入力欄下 |
| 「ログイン」ボタン | メール＋パスワードでログイン | フォーム下部 |
| 「Googleでログイン」ボタン | Google OAuth 認証 | ログインボタン下 |
| 「新規登録はこちら」リンク | 登録フォームへ切り替え | フォーム下部 |
| 「ログインはこちら」リンク | ログインフォームへ切り替え | フォーム下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| メール＋パスワードログイン | メール・パスワード入力欄＋「ログイン」ボタン | 入力後クリック → 認証 → P001（ホーム画面）へ遷移 |
| Google ログイン | 「Googleでログイン」ボタン | クリック → Google OAuth → P001（ホーム画面）へ遷移 |
| 新規登録 | メール・パスワード・ユーザー名入力欄＋「登録」ボタン | 入力後クリック → アカウント作成 → P001（ホーム画面）へ遷移 |
| フォーム切り替え | 「新規登録はこちら」「ログインはこちら」リンク | クリックでフォーム切り替え（ページ遷移なし） |
