# 仕様書: 猫図鑑画面

## 目的

全猫種を一覧で確認できる画面。毛色・模様・毛の長さで絞り込みができ、興味のある猫の詳細を確認できる。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 猫種一覧表示 | 全猫種をリスト形式（名前＋小さな写真）で表示する |
| 2 | フィルタリング | 毛色・模様・毛の長さのドロップダウンで絞り込む |
| 3 | 詳細表示 | 猫をタップしてP005（猫詳細画面）へ遷移する |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| フィルターエリア | 毛色・模様・毛の長さの3つのドロップダウン | 画面上部 |
| 猫種リスト | 名前＋小さな写真のリスト表示 | 中央〜下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 猫種一覧表示 | 猫種リスト | 画面表示時に全件ロード |
| フィルタリング | フィルターエリア（ドロップダウン） | 選択変更 → リストをリアルタイム絞り込み |
| 詳細表示 | 猫種リストの各アイテム | タップ → P005（猫詳細画面）へ遷移 |
