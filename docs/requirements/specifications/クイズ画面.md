# 仕様書: クイズ画面

## 目的

猫の種類を覚えるためのメイン学習画面。写真→種類名、種類名→写真の2方向で出題し、繰り返し学習できる。間違えた問題を優先して出題することで効率的な定着を促す。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 写真→種類名クイズ | 猫の写真を見て、4つの選択肢から種類名を選ぶ |
| 2 | 種類名→写真クイズ | 猫の種類名を見て、4つの選択肢から写真を選ぶ |
| 3 | 回答選択 | 4択の選択肢をタップして回答する |
| 4 | 進捗表示 | 現在の問題番号（例：3/10）を表示する |
| 5 | 優先出題 | 間違えた問題を優先して出題する |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| 問題番号 | 現在の問題番号（例：3/10） | 画面上部 |
| 問題エリア | 猫の写真 または 種類名テキスト | 中央上部 |
| 選択肢ボタン × 4 | 4つの回答候補 | 中央下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 写真→種類名クイズ | 問題エリア（写真）＋選択肢ボタン | 写真を見て種類名をタップ → P003（解説画面）へ遷移 |
| 種類名→写真クイズ | 問題エリア（テキスト）＋選択肢ボタン | 種類名を見て写真をタップ → P003（解説画面）へ遷移 |
| 進捗表示 | 問題番号 | 回答するたびに更新（1/10 → 2/10...） |
