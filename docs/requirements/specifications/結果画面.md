# 仕様書: 結果画面

## 目的

10問のクイズ完了後に表示される画面。正答率・覚えた種類数を表示し、継続学習を促す。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 結果表示 | 今回セッションの正答率・正解数・不正解数を表示する |
| 2 | 累計表示 | 端末に保存された累計の覚えた種類数を表示する |
| 3 | もう一度挑戦 | ホーム画面（P001）へ戻り、再度クイズを開始できる |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| 正答率 | 今回セッションの正答率（例：7/10 正解） | 中央上部 |
| 累計覚えた種類数 | これまでに正解した猫の種類数 | 中央 |
| 「もう一度挑戦」ボタン | ホーム画面へ戻るボタン | 下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 結果表示 | 正答率 | セッション完了時に自動表示 |
| 累計表示 | 累計覚えた種類数 | localStorage から読み込み自動表示 |
| もう一度挑戦 | 「もう一度挑戦」ボタン | クリック → P001（ホーム画面）へ遷移 |
