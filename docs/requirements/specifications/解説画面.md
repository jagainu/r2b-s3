# 仕様書: 解説画面

## 目的

クイズの回答後に表示される画面。正解・不正解に関わらず、その猫の特徴と似ている種類を提示することで「なぜこの種類か」を学べる。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 正解・不正解表示 | 回答が正解か不正解かを表示する |
| 2 | 猫の特徴表示 | 正解の猫の写真・種類名・毛色・模様・毛の長さを表示する |
| 3 | 似た種類表示 | 見た目が似ている猫の種類を一覧表示する |
| 4 | 次の問題へ | 次の問題（クイズ画面）へ進む |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| 正解・不正解バナー | 正解/不正解を視覚的に表示 | 画面上部 |
| 猫の写真 | 正解の猫の代表写真 | 中央上部 |
| 種類名 | 猫の種類名（正解） | 写真下 |
| 特徴リスト | 毛色・模様・毛の長さ | 種類名下 |
| 似た種類エリア | 似ている猫の種類名＋小さな写真 | 中央下部 |
| 「次の問題へ」ボタン | 次の問題に進むボタン | 画面下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 正解・不正解表示 | 正解・不正解バナー | 回答直後に自動表示 |
| 猫の特徴表示 | 猫の写真・種類名・特徴リスト | 自動表示 |
| 似た種類表示 | 似た種類エリア | 自動表示 |
| 次の問題へ | 「次の問題へ」ボタン | クリック → P002（クイズ画面）へ遷移、10問完了後はP006（結果画面）へ遷移 |
