- **開始時**: SNS で猫動画を見て楽しい気持ち

- **ブレークスルー**: クイズで不正解でも解説で「なるほど！」とわかる瞬間
- **ゴール達成時**: 友達との会話で種類を言い当てられてスッキリ・自信がつく
